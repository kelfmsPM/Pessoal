# Roadmap de Estudos

Visualização pessoal, em formato de mapa de metrô, dos temas que estou estudando/aprofundando — organizados por linha (categoria) e estação (tema), com status clicável (a estudar / estudando / concluído).

## Ver online

Depois do push, ative o **GitHub Pages** (Settings → Pages → Branch: main → pasta `/`) para ter uma URL pública tipo:

```
https://<seu-usuario>.github.io/roadmap-estudos/
```

## Estrutura

- `index.html` — página única (HTML/CSS/JS), sem dependências externas além de fontes do Google Fonts.

## Temas mapeados

- **Aquisição & Analytics** — Intercom, Pixel Google
- **Monetização** — Order Bump, UpSell, Adicional de juros (cadastro seller), Split de Pagamento
- **Experiência de Checkout** — Personalização Checkout (Banner, Cor de texto, Cronômetro, Prova Social), Tela de Recompensa
- **Integrações** — Sincronização Woocommerce
